# Customer Segmentation ML System

## Overview
This project builds a machine learning system that segments customers based on purchasing behavior using RFM analysis and clustering.

## Dataset
Online Retail dataset from UCI Machine Learning Repository.

## Feature Engineering
RFM (Recency, Frequency, Monetary) features were generated for each customer.

## Clustering Method
K-Means clustering was applied to identify customer segments.

## Elbow Method
The optimal number of clusters was determined using the elbow method.

## Results
Customers were segmented into 4 clusters representing different purchasing behaviors.
